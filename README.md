# COGS Lab Portfolio

> A hands-on portfolio documenting my work across cloud infrastructure, identity and access management, monitoring, incident response, and support operations.

## 👨‍💻 About

This repository contains the practical labs and documentation completed as part of the COGS training program. Each lab demonstrates real-world concepts through implementation, configuration, troubleshooting, and written analysis.

The objective of this portfolio is to showcase practical skills in:

- Cloud Infrastructure
- Linux Administration
- Networking Fundamentals
- Identity & Access Management (IAM)
- Authentication & Multi-Factor Authentication (MFA)
- Monitoring & Observability
- Incident & Change Management
- Documentation & Root Cause Analysis
- Git & GitHub Collaboration
